# Implementation Plan: Modern GUI Redesign

**Branch**: `001-modern-gui-redesign` | **Date**: 2026-04-18 | **Spec**: [spec.md](./spec.md)
**Input**: Feature specification from `specs/001-modern-gui-redesign/spec.md`

## Summary

Complete visual redesign of all 15 screens across kid and admin role groups. The redesign
introduces a new indigo + amber design system, replaces the existing #59f20d neon palette with a
more refined spiritual/educational aesthetic, adds shadcn/ui component primitives and Framer Motion
animations for polish, and ensures Arabic Bible text renders with a dedicated serif font. The
existing Next.js 16 App Router routes and Supabase data layer are fully preserved вЂ” only the
presentation layer changes.

## Technical Context

**Language/Version**: TypeScript 5 / Next.js 16.1.6 (App Router) / React 19.2.3
**Primary Dependencies**:
- Tailwind CSS v4 (existing) вЂ” design system tokens
- shadcn/ui (new) вЂ” accessible component primitives (Radix UI based)
- Framer Motion (new) вЂ” page transitions + micro-animations
- Lucide React (new, via shadcn) вЂ” icon system replacing emoji-only icons
- Cairo (Google Font, new) вЂ” Arabic serif font for Bible passages
- Plus Jakarta Sans (Google Font, new) вЂ” English UI typeface replacing Spline Sans
**Storage**: Supabase PostgreSQL (existing, unchanged)
**Testing**: Manual browser testing per constitution (no automated test framework configured)
**Target Platform**: PWA / mobile-first web (iOS Safari, Android Chrome); desktop secondary
**Project Type**: Web application (Next.js fullstack, redesign of UI layer only)
**Performance Goals**: LCP < 3s on 4G; no layout shift from fonts (next/font with display:swap);
  Server Components by default; `"use client"` only where interactivity requires it
**Constraints**: Arabic RTL rendering; offline PWA (service worker unchanged); Supabase auth
  cookie session unchanged; 15 screens in scope; no route changes; no data layer changes
**Scale/Scope**: ~15 screens, ~20 new/rewritten components; 3 roles (kid, admin, superuser)
